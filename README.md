## 🤖 What Is This?

An AI-powered REST API that generates **Mr. Robot** style quotes on demand.
Every quote is **freshly generated by AI** — no hardcoded quotes.

```
guest@fsociety:~$ curl https://quotes.adhikariashwin0.com.np/api/quote/random
```

```json
{
  "success": true,
  "quote": {
    "text": "The system was never broken. It was built this way.",
    "character": "Elliot Alderson",
    "category": "society"
  },
  "generated": "fresh"
}
```

---

## 🔗 API Endpoints

| Endpoint | Description | Response |
|----------|------------|----------|
| `GET /api/quote/random` | Fresh AI-generated quote every request | New quote each time |
| `GET /api/quote/daily` | Quote of the day (cached for 24hrs) | Same quote all day |
| `GET /api/quote/svg` | Generate SVG from the quotes | parameters: mode=random/daily theme=[Available Themes](#themes)|

### Base URL

```
https://quotes.adhikariashwin0.com.np
```

## ⚡ Quick Start

### Get a Random Quote

```bash
curl https://quotes.adhikariashwin0.com.np/api/quote/random
```

### Get Daily Quote

```bash
curl https://quotes.adhikariashwin0.com.np/api/quote/daily
```
### Get SVG Quote

```bash
curl https://quotes.adhikariashwin0.com.np/api/quote/svg?mode=daily&theme=mrrobot
```

### Use in JavaScript

```javascript
fetch('https://quotes.adhikariashwin0.com.np/api/quote/random')
  .then(res => res.json())
  .then(data => {
    console.log(data.quote.text);
    console.log(`— ${data.quote.character}`);
  });
```

### Use in Python

```python
import requests

response = requests.get('https://quotes.adhikariashwin0.com.np/api/quote/random')
data = response.json()

print(data['quote']['text'])
print(f"— {data['quote']['character']}")
```

## 📦 Response Format

```json
{
  "success": true,
  "quote": {
    "text": "Every firewall is a dare. Every encrypted file, a love letter waiting to be read.",
    "character": "Elliot Alderson",
    "category": "hacking"
  },
  "generated": "fresh"
}
```

### Fields

| Field | Type | Description |
|-------|------|-------------|
| `success` | boolean | Whether the request succeeded |
| `quote.text` | string | The AI-generated quote |
| `quote.character` | string | Mr. Robot character who "said" it |
| `quote.category` | string | Quote category (hacking, society, etc.) |
| `generated` | string | `fresh` = new from AI, `cached` = daily cache |

## 🎯 Add the quotes in Github profile README

Add this to your GitHub Actions workflow to auto-update your README with daily quotes:

```yaml
name: Update Mr Robot Quote
on:
  schedule:
    - cron: '0 0 * * *'
  workflow_dispatch:

jobs:
  update-quote:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v3
      - name: Fetch and Update Quote
        run: |
          RESPONSE=$(curl -s https://quotes.adhikariashwin0.com.np/api/quote/daily)
          python3 << 'EOF'
          import json, re, os
          from datetime import datetime

          # Fetch text quote
          response = os.popen('curl -s https://quotes.adhikariashwin0.com.np/api/quote/daily').read()
          data = json.loads(response)
          q = data['quote']
          formatted = f'> *"{q["text"]}"*\n>\n> — **{q["character"]}** `#{q["category"]}`'

          with open('README.md', 'r') as f: content = f.read()

          # Update text quote between tags
          content = re.sub(
              r'<!-- MR_ROBOT_QUOTE_START -->.*?<!-- MR_ROBOT_QUOTE_END -->',
              f'<!-- MR_ROBOT_QUOTE_START -->\n{formatted}\n<!-- MR_ROBOT_QUOTE_END -->',
              content, flags=re.DOTALL)

          # Update SVG cache-buster with today's date
          today = datetime.now().strftime('%Y%m%d')
          def update_svg_url(match):
              url = match.group(0)
              url = re.sub(r'[&?]cachebust=\d+', '', url)
              separator = '&' if '?' in url else '?'
              return f'{url}{separator}cachebust={today}'

          content = re.sub(
              r'https://quotes\.adhikariashwin0\.com\.np/api/quote/svg[^\s"\')]*',
              update_svg_url, content)

          with open('README.md', 'w') as f: f.write(content)
          EOF
      - name: Commit
        run: |
          git config user.email "action@github.com"
          git config user.name "GitHub Action"
          git add README.md
          git diff --staged --quiet || git commit -m "🤖 Daily Quote Update"
          git push
```

Add this code on README.md to generate quote

- For `text` quote:
  ```
    ## 🤖 Mr. Robot Quote of the Day

    <!-- QUOTE_START -->
    > Quote loads here automatically every day
    <!-- QUOTE_END -->
  ```
- For `SVG` quote (added automatically via github workflow don't need code below):
  ```
    <img src="https://quotes.adhikariashwin0.com.np/api/quote/svg?theme=default&mode=daily" />
  ```
